# 🏥 smart hospital management system (shms)

a smart hospital management system designed to streamline patient flow, reduce waiting time, and improve resource management across hospitals.

🔗 live: https://shms-latest-app.vercel.app/

## ✨ technologies

- react  
- typescript  
- css  
- firebase (firestore)  
- ai-assisted development  

## 🚀 features

- patient queue and appointment management  
- role-based access control (admin, staff, users)  
- real-time data synchronization using firestore  
- smart hospital workflow simulation  
- responsive frontend with structured ui  
- scalable system for future city-wide integration

## 📍 the process

this project was built as a team project where i worked as the project lead, coordinating development and ensuring smooth integration between frontend and backend.

we focused on solving real-world hospital challenges like long waiting times and inefficient resource usage. the system was designed to simulate a smart workflow where patient handling and hospital operations can be managed more effectively.

firebase firestore was used to manage real-time data, while ai tools were used to accelerate development and experimentation.
